# Práctica 1 - CREACIÓN DE CLASES Y ENCAPSULAMIENTO EN PHP

## Objetivo
Comprender el uso de clases en PHP mediante la creación de una clase que implemente atributos privados, constructor, getters y setters.

## Descripción de la clase
La clase Usuario representa un usuario con dos atributos:

- nombre
- correo

Los atributos son privados y se accede a ellos mediante métodos getters y setters.

La clase incluye:
- Constructor para inicializar los atributos
- Métodos getNombre() y getCorreo()
- Métodos setNombre() y setCorreo()

## Instrucciones de ejecución

1. Abrir la carpeta del proyecto en Visual Studio Code.
2. Ejecutar el archivo `index.php`.
3. El programa crea un objeto Usuario y muestra su nombre y correo utilizando los métodos getters.
